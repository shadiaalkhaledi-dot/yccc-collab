## Managing the Team

**Add someone:**
1. Add a row to Active Team above
2. Invite on GitHub: repo → Settings → Collaborators → Add people
3. Note in HANDOFF.md

**Remove someone:**
1. Move their row to Alumni with leave date
2. Remove GitHub access: repo → Settings → Collaborators → remove
3. Note in HANDOFF.md

**Update GitHub username:**
Edit the GitHub column directly.

**Update AI tool:**
Edit the AI Tool column. History is preserved via git commits.
