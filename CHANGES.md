## How This Works

A Google Apps Script runs on each tracked Google Doc. When anyone saves changes, the script automatically appends a row to this table with:
- Who made the edit (Google account name)
- What time
- A summary of what section changed

No manual action needed. Just edit the doc normally.

See `APPS_SCRIPT.js` for the script code to install on each Google Doc.
